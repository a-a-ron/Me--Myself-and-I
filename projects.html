Rails Project

I'm currently working on building a rails application for a local bike shop. Being my first offical rails project, i'm going through the book, Rails 4 in Action by Ryan Bigg, Yehuda Katz and Steve Klabnik. It's been extremely helpful and informative and I highly recommend reading this book if you have an interest in learning ruby on rails.

It's been fun to learn about popuar Ruby Gems used for Rails projects. Here is a list of some i'll be using for my bike application:

  • Devise - A felxible authentication solution for Rails based on Warden. It encrypts ad stores a password in the database to validate the authenticity of a user while signing in. The authentication can de done through POST rewquests or HTTPS Basic Authentication.
  • CanCan - An authorization library for Ruby which restricts that resources a given user is allowed to access. All permissions are defined in a single location( the ability class) and not duplicated across controllers, views and database queries.
  • Rspec - Replaces manual testing with automated testing. Refered to as red-green-refractor because you write code known as specs that contain examples that have not been created yet and then write the code to make the test pass.
  • Capybara - a browser simulator in Ruby that is used for integration testing along with Rspec. This kind of testing ensures that when a link is clicked in your application, it goes to the correct page, or when you fill in a form and click the Submit button, an onscreen message tells you that the form's operation was successful. In other words, it simulates how a user would interact within an application.
  • Factory_girl - Provides an easy way to use factories to create new objects for your tests. Factories define a bunch of default values for an object, allowing you to easily craft example objects you can use in your tests.
  • Twilio - A helper library that lets you write Ruby code to make HTTP requests to the Twilio API. Used to contact customers or users to my application by ending emails or text notifictaions when their order or repair is complete.
